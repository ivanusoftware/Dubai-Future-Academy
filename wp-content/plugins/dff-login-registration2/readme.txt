Just another Custom Registration and Login Forms plugin with New Recaptcha. Simple but flexible.

== Description ==

Custom Registration and Custom Login Forms plugin is simple plugin to show custom registration and custom login on any post or pages with just short code like [dff_login_form] and [dff_register_form].

== Installation ==

1. Upload the entire `custom-registration-and-login-forms-with-new-recaptcha` folder to the `/wp-content/plugins/` directory.

1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Register Plugin' section at leftside in your WordPress admin panel.
